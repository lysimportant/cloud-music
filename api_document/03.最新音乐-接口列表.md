# 最新音乐 接口文档

>  所有接口的请求方式都是`get`

## 最新音乐

说明 : 调用此接口 , 可获取最新音乐

**必选参数 :**

`type`: 地区类型 id,对应以下:

```
全部:0

华语:7

欧美:96

日本:8

韩国:16
```

**接口地址 :**  https://autumnfish.cn/top/song





## 播放歌曲

> 说明 : 调用此接口 , 传入音乐 `id`, 可获得歌曲播放地址

**参数 :** `id`: 音乐 id, 如 `id=347230`

**接口地址 :** https://autumnfish.cn/song/url
