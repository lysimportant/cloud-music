# 最新MV页面 接口文档

>  所有接口的请求方式都是`get`

## 搜索接口

> 根据关键字搜索结果，通过调整类型，搜索不同的值

**必选参数 :** `keywords` : 关键词

**可选参数 :** 

1. `limit` : 返回数量 , 默认为 30 
2. `offset` : 偏移数量，用于分页 , 如 : 如 :( 页数 -1)*30, 其中 30 为 limit 的值 , 默认为 0

3. `type`: 搜索类型；默认为 1 即单曲 , 取值意义 :如下

```
歌曲:1
歌单:1000
MV:1004
```

**接口地址 :**  https://autumnfish.cn/search
